# argos-to-sql
Bash script that reads argos export files and creates a sql file for each report in the datablock.

# Parameters
  -- .argosexport file
# Output
  -- .argos.sql file for each report in the datablock
# Dependencies
  -- GNU awk 3.1.7+
  -- GNU grep 2.6.3+
  -- csplit (GNU coreutils) 8.4+
